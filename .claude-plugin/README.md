# Metaxy Claude Code Plugin

A Claude Code plugin for [Metaxy](https://anam-org.github.io/metaxy/), a feature metadata management system for multimodal pipelines.

## Installation

Add the marketplace and install:

```bash
/plugin marketplace add anam-org/metaxy
/plugin install metaxy
```

## Features

- **`/metaxy` skill**: Guidance on feature definitions, versioning, metadata stores, and testing
- **MCP server**: Exposes feature graph and metadata store tools to Claude

## Documentation

See the [Claude Code integration docs](https://anam-org.github.io/metaxy/integrations/ai/claude/) for detailed usage.

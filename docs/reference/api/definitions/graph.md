# Feature Graph

`FeatureGraph` is a global "God" object that holds all the features loaded by Metaxy via the [feature discovery](../../../guide/concepts/feature-discovery.md) mechanism.

Users may interact with `FeatureGraph` when writing custom migrations, otherwise they are not exposed to it.

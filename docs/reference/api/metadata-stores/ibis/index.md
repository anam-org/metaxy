# Database Metadata Stores

Metadata stores that are backed by databases are implemented with [Ibis](https://ibis-project.org/).

Users can extend [IbisMetadataStore][metaxy.metadata_store.ibis.IbisMetadataStore] to work with databases that are not natively supported by Metaxy.

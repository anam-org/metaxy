# Integrations

API docs for Metaxy integrations with third-party tools.

- The [SQLModel](sqlmodel.md) integration allows coupling Metaxy feature definitions with `sqlalchemy` tables.

- The [SQLAlchemy](sqlalchemy.md) integration provides convenience helpers for retrieving `sqlalchemy.MetaData` objects. These can be used to set up `alembic` migrations for Metaxy features backed by `sqlalchemy`.
